# Bill-splitting-calculator-python
CLI-based bill splitting calculator built in Python.

This is my first Python project. It is a simple CLI-based application that helps calculate and split a bill.

## 📌 Features
- Calculates total bill including tax and tip
- Displays a structured bill summary
- Supports equal bill splitting
- Supports unequal bill splitting based on individual contributions

## ⚙️ How it works
- User enters the original bill amount
- Selects tax percentage and tip percentage
- Program calculates total amount
- User can choose to:
  - View final amount
  - Split the bill equally
  - Split the bill unequally

## ▶️ How to run
1. Make sure Python is installed
2. Download or clone this repository
3. Run the file: python bill_splitting_calculator.py
